Switch Button Plugin

Switch Button is a jQuery UI widget that mimics the iPhone style toggle for on and off states.

Plugin Options

You can set any of the following options to customize it to fit your needs:


// default options
$('selector').switchbutton({
  state: 'off'
});

NOTE: Options can be set using the data-options attribute as well, and passing it a switchbutton object. For example:


<div data-role="switchbutton" data-options='{ "switchbutton": { "state": "on" }}'>
  <input type="radio" id="switch-off" name="switch-radio" checked="checked">
  <label for="switch-off">Off</label>

  <input type="radio" id="switch-on" name="switch-radio">
  <label for="switch-on">On</label>
</div>

CSS Hooks

  • On Checkbox: .ui-switch-button-on
  • Off Checkbox: .ui-switch-button-off
